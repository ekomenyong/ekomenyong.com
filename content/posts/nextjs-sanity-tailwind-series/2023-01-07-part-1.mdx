## Setup Next.js and TypeScript

Let's use `create next-app` to initialize a new Next.js project that uses the `app` directory. I'm using `yarn`, but feel free to use the package manager of your choice.

```bash title="Terminal"
npx create-next-app@latest --experimental-app
# or
yarn create next-app --experimental-app
# or
pnpm create next-app --experimental-app
```

## Install Tailwind CSS

```bash title="Terminal"
yarn add -D tailwindcss postcss autoprefixer @tailwindcss/typography @tailwindcss/forms
yarn add tailwind-merge clsx
npx tailwindcss init -p
```

### Setting up `clsx` and `tailwind-merge`

```ts title="utils/clsxm.ts"
import clsx, { ClassValue } from "clsx";
import { twMerge } from "tailwind-merge";

export function clsxm(...classes: ClassValue[]) {
  return twMerge(clsx(...classes));
}
```

### ESLint Prettier, Husky, and CommitLint

```bash title="Terminal"
yarn add -D prettier prettier-plugin-tailwindcss pretty-quick husky @commitlint/cli @commitlint/config-conventional
```

```json title="/.commitlintrc.json"
{
  "extends": ["@commitlint/config-conventional"]
}
```

```bash title="Terminal"
yarn husky install

npx husky add .husky/commit-msg  'npx --no -- commitlint --edit ${1}'

npx husky add .husky/pre-commit  'npx pretty-quick --staged'

npx husky add .husky/post-merge 'yarn'
```
